# AWS Cloud Practitioner – Knowledge Checks (KCs)

Este repositório reúne **anotações, resumos e atividades de estudo** desenvolvidos durante a formação **AWS re/Start**, por meio da **Escola da Nuvem**, com foco na preparação para a certificação **AWS Certified Cloud Practitioner**.

Os materiais aqui organizados correspondem aos **Knowledge Checks (KCs)** — atividades conceituais utilizadas ao longo do programa para reforçar fundamentos de computação em nuvem, serviços da AWS, segurança, arquitetura e boas práticas.

> ⚠️ **Nota importante**  
> Este repositório tem finalidade **educacional e pessoal**, servindo como registro de aprendizado.  
> Não substitui materiais oficiais da AWS nem representa conteúdo confidencial ou proprietário.
